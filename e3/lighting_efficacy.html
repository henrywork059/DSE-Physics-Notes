E3 — Lighting: Efficacy (lm W⁻¹)

HKDSE Physics — Elective E3: Energy & Use of Energy · Use luminous efficacy to compare lamps fairly: “How much visible light (lm) do I get per watt?”

← Back to index ← Previous Next →

For lamp questions, brightness is linked to luminous flux \(\Phi\) (lm), but electricity use depends on input power \(P\) (W).
Efficacy connects them: \(\text{Efficacy} = \dfrac{\Phi}{P}\) (lm W\(^{-1}\)).

Exam focus: \(\text{Efficacy} = \Phi/P\), \(\Phi = (\text{efficacy})\times P\), compare energy-saving lamps Skills: unit check (lm, W), interpret packaging / labels, compare “same brightness” cases

1. Syllabus focus (what you must be able to do)

  • Define efficacy of electric lights as a ratio of luminous flux to electrical power input.
  • Use \(\text{Efficacy} = \Phi/P\) (and \(\Phi = (\text{efficacy})\times P\)) to solve numerical problems.
  • Compare lamps and justify which is more energy-saving for the same brightness requirement.
Idea
A lamp can have a high power rating but still look dim. In lighting, it is better to compare using \(\Phi\) (lm) and efficacy (lm W\(^{-1}\)).

2. Definition & units

What efficacy means

  • Luminous efficacy (or simply efficacy) measures how efficiently a light uses electrical energy for illumination.
  • It links luminous flux \(\Phi\) (lm) and input power \(P\) (W).

Formula

  • \(\displaystyle \text{Efficacy} = \frac{\Phi}{P}\)
  • Rearrange when needed: \(\displaystyle \Phi = (\text{efficacy})\times P\)

Unit

  • \(\Phi\) in lumen (lm), \(P\) in watt (W) \(\Rightarrow\) efficacy in \(\text{lm W}^{-1}\).
Unit trap
Use watt (W), not kW, unless you convert correctly. Always state the unit \(\text{lm W}^{-1}\).

3. How to compare lamps (exam logic)

Case A: “Same brightness” (same luminous flux \(\Phi\))

  • If two lamps have the same \(\Phi\), then the one with smaller input power uses less electricity.
  • Because \(\text{efficacy} = \Phi/P\), smaller \(P\) for the same \(\Phi\) means higher efficacy.

Case B: “Same power rating” (same \(P\))

  • If two lamps have the same \(P\), the one with higher efficacy has larger luminous flux, so it appears brighter.
  • Reason: \(\Phi = (\text{efficacy})\times P\).
Exam wording
“The LED lamp is more energy-saving because it consumes less power to give the same luminous flux (i.e. higher efficacy).”
Concept link
Luminous flux \(\Phi\) already includes correction for the human eye’s sensitivity to different wavelengths. So it is a good measure for “how bright it looks”.

4. Worked examples (typical DSE style)

Example 1: Find luminous flux from efficacy and power

  • Given an LED lamp: efficacy = \(110\,\text{lm W}^{-1}\), input power = \(6.5\,\text{W}\)
  • \(\Phi = (\text{efficacy})\times P = 110\times 6.5 = 715\,\text{lm}\)
  • Answer: \(\boxed{715\,\text{lm}}\)

Example 2: Find efficacy from luminous flux and power

  • Suppose a 60-W GLS (incandescent) lamp gives \(\Phi = 715\,\text{lm}\).
  • \(\text{Efficacy} = \dfrac{\Phi}{P} = \dfrac{715}{60} = 11.9\,\text{lm W}^{-1}\)
  • Answer: \(\boxed{11.9\,\text{lm W}^{-1}}\)
Common mistake
Don’t write “lm/W” for luminous flux. Luminous flux is just lm. Only efficacy is \(\text{lm W}^{-1}\).
Link to cost
If two lamps have the same brightness (same \(\Phi\)), the one with lower \(P\) uses less electrical energy \(E=Pt\), so the running cost is lower.

5. Typical efficacy values (reference)

  • Different lamp types have very different efficacies.
  • In general: LED and fluorescent lamps have much higher efficacy than incandescent lamps.
Lamp type Typical efficacy Comment
Incandescent (GLS) 10–20 lm W\(^{-1}\) Low efficacy because much energy becomes heat (infra-red).
CFL / fluorescent 30–100 lm W\(^{-1}\) Higher efficacy than incandescent for similar brightness.
LED 30–100 lm W\(^{-1}\) (often high) High efficacy, commonly used for energy-saving lighting.
Know more (optional)
The maximum possible efficacy is \(683\,\text{lm W}^{-1}\) for an ideal light source emitting only 555 nm (green‑yellow) light and converting all electrical energy to light.

6. Reminders & common exam traps

  • Don’t confuse power rating with brightness. Brightness is linked to \(\Phi\) (lm).
  • When asked “most energy-saving for the same brightness”, choose the lamp with highest efficacy (or lowest \(P\) for the same \(\Phi\)).
  • State the unit clearly: efficacy in \(\text{lm W}^{-1}\).
  • If colours/wavelengths are involved, remember \(\Phi\) is corrected for the human eye’s sensitivity.
Quick check
If a lamp has \(\Phi = 2700\,\text{lm}\) and \(P = 45\,\text{W}\), then efficacy \(= 2700/45 = 60\,\text{lm W}^{-1}\).

7. Key terms

  • Luminous flux \(\Phi\): total light output corrected for human vision (unit: lm).
  • Input power \(P\): electrical power consumed by the lamp (unit: W).
  • (Luminous) efficacy: \(\Phi/P\) (unit: \(\text{lm W}^{-1}\)).
  • Energy-saving lamp: a lamp with higher efficacy (needs less power for the same \(\Phi\)).
